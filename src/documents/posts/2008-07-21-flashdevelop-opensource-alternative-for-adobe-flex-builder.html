--- layout: default title: FlashDevelop - OpenSource alternative for Adobe Flex Builder created: 1216625100 ---

FlashDevelop is an open source project, created for AS2 and AS3 projects. It’s an IDE which can be a good alternative for a Flex Builder from adobe. The main advantage is that it’s free! It’s also a lot lighter. It has a code autocompleter, panels for presenting errors, warnings and tasks. It’s very like Flex Builder, but it doesn’t have visual editor.
