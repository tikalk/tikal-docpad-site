Introducing the JBoss Seam framework - an innovative new framework for web application development with the Java EE 5.0 platform. Seam brings two new standards, Java Server Faces (JSF) and EJB 3.0, much closer together. In addition we'll show Seam’s automatic persistence context management, and the first-class constructs it provides for the definition of conversations in the application flow..
